## 为什么自建

- **可控性**：版式、域名、内容完全由自己掌握。
- **永久存档**：不依赖第三方平台的存续策略。
- **展示项目**：可以同时承载博客与作品集。

## 技术选型

这个站点使用 [Astro](https://astro.build) 构建，托管在 GitHub Pages 上。下面是一段配置示例：

```javascript
import { defineConfig } from 'astro/config';

export default defineConfig({
  site: 'https://example.github.io',
  base: '/',
});
```
