#### 1\. 将fasta中的header name替换为对应的fasta文件名

```
#输出在屏幕，不修改原文件
awk '/^>/{print ">" substr(FILENAME,1,length(FILENAME)-6); next} 1' *.fasta
#直接修改原文件
for file in *.fna_16s; do sed -i "s/>.*/>${file%%.*}/" "$file" ; done
#${file%%.*}中的%%表示file中只保留.左边的字符串

```

