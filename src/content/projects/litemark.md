## 概述

LiteMark 是一个跨平台的 Markdown 编辑器桌面应用，使用 Tauri 作为后端，结合现代前端实现实时预览与流畅的写作体验。

## 特性

- **原生性能**：Rust 核心，启动快、内存占用低。
- **实时预览**：所见即所得的编辑体验。
- **多平台**：Windows / macOS / Linux 一致体验。
